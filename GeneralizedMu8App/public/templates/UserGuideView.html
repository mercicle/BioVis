Welcome to the Mu-8 User Guide

So how do you use Mu-8? Well, to start, Mu-8 takes two inputs:

1. Multiple Sequence Alignment file (created from, e.g., clustal output)
2. 3D Coordinates file (created from the PDB file).

You upload these files using the "New Sequence Analysis" button on the index page after logging in. When you submit a new sequence analysis, Mu-8 will then use your sequences and 3-dimensional structure to construct the visualization. To learn how to use and interpret all of the visual encoding and features of Mu-8, please see our paper: Mu-8: Visualizing Differences between a Protein and its Family
