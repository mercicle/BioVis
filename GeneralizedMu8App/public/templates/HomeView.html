Mu-8

Mu-8 is a novel, interactive visualization tool for comparing proteins to their family and for identifying potential regions that cause functional breakdown. This framework provides a complete quantitative landscape of residue characteristics in the context of both primary and tertiary structure.

To try out the lasted version click Try Demo below. The latest version requires some time to load, so please be patient. To try the classic version, which loads faster, click Try Classic Demo.

If you want to upload your own data, please create an account and log in.

Mu-8 was created as a contribution to the 2013 BioVis Contest. The demo versions use the dataset provided by Drs. Magliery and Sullivan at The Ohio State University.
